# Right-Triangle-Calculator
A simple calculator for determining area and perimeter of a right triangle. This was made using HTML and JavaScript with no CSS involved. The user inputs two values, and the calculator makes computations assuming we are solving for a right triangle.

The calculator gives two options at first: "Compute Perimeter" and "Other options. "Compute Perimeter" will take the two inputs from the user and determine the perimeter. "Other Options" will create 3 new buttons that the user can use.

After pressing "Other Options", three new buttons should appear labeled "Compute Area", "Compute Both", and "Clear Results". "Compute Area" will compute the area of the triangle given the two user inputs. "Compute Both" will compute both the area and perimeter. "Clear Results" will clear the computation results from the webpage. 

You may notice that the "Clear Results" button activates as soon as the cursor touches the button. This is intentional, and showcases the use of event handling within the program. In this case, I used the `.observe` method for determining if the cursor touches the button.
